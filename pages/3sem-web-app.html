Web-App - Laundry Helper - kun mobil

3. semester - individuel projekt

Hvad gik det ud på?

Projektet gik ud på at udarbejde en Web App, singel page aplication ved brug af HTML, CSS, JS, variabler, objects, arrays og JSON data

Arbejdsprocessen

Arbejdsprocessen startede med en intervieve nogle unge i målgruppen, efterfulgt af meget analysearbejde både om målgruppen og konkurenter. Under designfacen og kodingsfacen benyttede jeg mig af flere typer brugertest både high og low fidelity tests.

Hvad fik jeg ud af det?

Det jeg fik ud af dette projekt var, at arbejde med en singel page application og arbejde med JSON da dette er nyt stoff. Det var derfor svært men meget lærrerigt.
